DesyreAI is an LLM-powered AI assistant designed to handle everyday productivity tasks and automate routine digital activities.

⚠️ Status: Currently in development.

The goal of DesyreAI is to function as a personal AI assistant capable of interacting with different tools and services to help users manage their daily workflows more efficiently.

## Overview

DesyreAI uses a Large Language Model (LLM) to understand user requests and perform tasks automatically. Instead of only answering questions like a typical chatbot, DesyreAI is designed to take actions on behalf of the user.

These actions may include managing schedules, sending emails, retrieving information from the internet, and performing other automated tasks.

## Current Capabilities

DesyreAI is being built to support the following features:

Email automation (send emails on behalf of the user)

Scheduled tasks

Calendar management

Internet search and information retrieval

YouTube video search

General conversational assistance

DesyreAI can help users with everyday tasks such as:
